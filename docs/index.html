Properties

  • string url

    Target url

  • string | object data

    Ajax payload

  • string method

    GET|POST|DELETE|PUT etc

  • object headers

    Headers to add to XHR object:
    Header-Name: header-value

  • string username

    XHR username

  • string password

    XHR password

  • string dataType

    html - along with selector option treats response as html, creates a document out of it and returns selected element.
    xml - parse response as xml and return element(s) using selector option
    json parse response as json and return the resulting object
    fragment - turn response into a DocumentFragment
    script - evaluate response as a script

  • int timeout

    Abort on timeout

  • string contentType

    Request content type. Set contentType: json to transform data into json automatically and set header to text/plain.

  • string contentTypeHeader

    If contentType = json, set this to specific header you want to send

  • object xhrFields

    Key:value pairs to set to xhr object

  • boolean jsonp

    Make a jsonp request

  • string jsonParam

    • @default callback

    Name of the parameter with callback function name: url?=

  • string jsonpCallback

    Name of the callback function in global scope

  • string transport

    iframe|xhr|script
    If files or form options are set and browser doesn't support FormData, transport will be set to iframe.

  • boolean replace

    When using ajax.load(el, url, opt) if replace=true, all contents of el will be replaced by response;
    if replace=false, response will be appended.

  • string selector

    See dataType

  • FormElement form

    Souce of request data and files, target url and request method
